# SorteerAlgoritmes

## ArrayUtils
Implementeer de methodes in de klasse `ArrayUtils` zoals beschreven in de cursus. Volg hierbij ook nauwgezet de cursus.

De klasse `ArrayUtils` wordt gebruikt in alle andere klassen, het is dus belangrijk om de ontbrekende methodes correct te implementeren.

## Iteratieve sorteeralgoritmen
- Selection sort: `SelectionSorter`
- Bubble sort (geoptimaliseerd): `BubbleSorter`
- Insertion sort: `InsertionSorter`

## Recursieve sorteeralgoritmen
- Quick sort: `QuickSorter`
- Merge sort: `MergeSorter`

## Vergelijk sorteeralgoritmen
De `CompareSortingAlgoritmes` klasse bevat een uitvoerbare `main()` methode die willekeurige 
niet-gesorteerde arrays van integers zal aanmaken van verschillende grootte en zal 
die sorteren met behulp van de eerder bestudeerde sorteeralgoritmen. 
De tijd die hiervoor nodig is wordt gelogd.
